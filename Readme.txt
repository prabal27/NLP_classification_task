1.Following packages are necessary to run this program:
 numpy
 pandas
 tensorflow
 keras
 re
 nltk
 sklearn
 
2.Model file is stored as : my_model.h5

3.In case you want to re-train the model, delete the model file "my_model.h5". Program will automatically retrain the model and save the model file.(use keras-gpu if you have nvidia gpu to speed up the process)

4.After the model file is loaded, ask the question and it will return you the intent of the question.

5.Type "exit" to end the program
